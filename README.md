#BluePlay

A clean, lightweight browser gaming hub built for modern browsers and hosted entirely on GitHub Pages.

Playgrid is a fast, minimal gaming platform focused on smooth UI, simple access, and distraction-free gameplay. The project is fully frontend-based and designed to run without any backend or server dependencies.

---

## Overview

Playgrid is an open-source browser game hub built using only HTML, CSS, and JavaScript. The goal is to provide a responsive and visually polished experience while keeping performance high and code structure simple.

Features include:

- Dark themed interface with blue accent styling
- Animated rain background rendered with Canvas
- Local username system (no accounts required)
- Responsive game grid layout
- Embedded browser games
- Clean and minimal UI animations
- Static hosting compatible
